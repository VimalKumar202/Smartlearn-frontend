SmartLearn is an AI-Powered Student Learning Platform designed to make education smarter and more interactive.
It helps students learn through AI-generated notes, quizzes, and personalized study support.
Teachers can upload study materials, track student performance, and manage classes easily.
The platform includes chatbot assistance, progress monitoring, and role-based access for students, teachers, and admins.
SmartLearn aims to improve learning efficiency using modern technology and artificial intelligence
